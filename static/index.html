API Simples e Poderosa
Uma chamada para classificar, extrair e validar. Resposta sync ou via webhook.
  Request
# Pipeline completo: classifica + extrai curl -X POST https://sua-api.com/process \ -H "X-API-Key: $API_KEY" \ -F "arquivo=@documento.jpg" # Apenas classificar curl -X POST https://sua-api.com/classify \ -H "X-API-Key: $API_KEY" \ -F "arquivo=@documento.jpg" # OCR generico (PDF/imagem) curl -X POST https://sua-api.com/ocr \ -H "X-API-Key: $API_KEY" \ -F "arquivo=@contrato.pdf"

Integracao Flexivel
Dois modos de entrega para se adaptar ao seu fluxo.
⏳

Modo Sincrono

Envie o documento e receba o resultado na mesma requisicao. Ideal para fluxos interativos onde o usuario aguarda o processamento.

POST /process
200 OK + resultado completo
🔔

Modo Webhook

Receba um 202 imediatamente e o resultado sera enviado via POST para sua URL quando pronto. Ideal para processamento em massa.

POST /process?delivery_mode=webhook
202 Accepted + callback quando pronto
